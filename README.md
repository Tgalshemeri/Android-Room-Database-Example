# Android Room Database Example in Kotlin

First you to need add to your app's build.gradle 
```
apply plugin: 'kotlin-kapt'
```

```
    implementation 'androidx.room:room-runtime:2.1.0'
    kapt 'androidx.room:room-compiler:2.1.0'
    
```
